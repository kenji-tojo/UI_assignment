Goal: Connect 4 corners by moving branches

How to use:

- Select a branch by clicking it.

- Move the branch by Dragging.

- Rotate the branch by Dragging with Shift key.

- You can also use Enter key to choose a branch and the arrow keys with/without Shift key to move/rotate the branch

Colors:

- Green dots: valid joints. Red dots: invalid joints

- Invalid group: Red branches. Valid group: other colors.
