Connect 4 corners by moving branches

How to use:

- Click: select a branch

- Drag: move the branch

- Shift + Drag: rotate the branch

- You can also use Enter key to choose a branch and the arrow keys with/without Shift key to move/rotate the branch.

Colors:

- Green dots: valid joints. Red dots: invalid joints.

- Each group of connected branches are shown in the same color.

Goal:

- Connect (bridge) the four corners by five branches only with green joints (red joints are not allowed).

- Try to make as many green joints as possible to get a high score.
